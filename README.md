# 🎵 LRC Lyrics Collection

A personal collection of synchronized lyrics files (`.lrc`) for various songs, created and shared for use in Stanza.

## 📖 About

This repository contains `.lrc` (LyRiCs) files — plain text files that store song lyrics synchronized with timestamps. These files allow compatible music players to display lyrics in sync with the audio playback, similar to karaoke.

All lyrics files in this repository were created/edited manually using Stanza and exported as `.lrc` files.

## 📁 Repository Structure

```
.
├── README.md
└── lyrics/
    ├── Artist Name - Song Title.lrc
    ├── Artist Name - Another Song.lrc
    └── ...
```

Files are named using the format: `Artist - Song Title.lrc`

## 🎧 What is an LRC File?

An LRC file is a simple text file containing lyrics with time tags. Each line typically looks like this:

```
[00:12.34]This is the first line of lyrics
[00:15.67]This is the second line
[00:19.20]And so on...
```

The time tags `[mm:ss.xx]` tell the music player exactly when each line should appear on screen.

## 🚀 How to Use

### Option 1: Download a Single File
1. Navigate to the `lyrics/` folder
2. Click on the `.lrc` file you want
3. Click the **Download raw file** button (the download icon)
4. Import it into Stanza App and have fun

### Option 2: Clone the Entire Repository
```bash
git clone https://github.com/M5Abed/Stanza-lrc.git
```

### Option 3: Download as ZIP
Click the green **Code** button at the top of the repo → **Download ZIP**

## 🤝 Contributing

This is a personal collection shared with friends, but contributions are welcome!

If you'd like to add or improve a lyrics file:
1. Fork this repository
2. Add or edit the `.lrc` file in the `lyrics/` folder
3. Make sure the filename follows the `Artist - Song Title.lrc` format
4. Submit a pull request

## ✅ Quality Guidelines

When contributing, please make sure:
- Timestamps are accurate and well-synced
- Lyrics are spelled correctly
- File encoding is **UTF-8** (especially important for non-English lyrics)
- The filename matches the `Artist - Song Title.lrc` convention
